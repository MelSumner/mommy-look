mommy-look
==============================================================================

Adds the "mommy-look" icon as a component to your project.


Compatibility
------------------------------------------------------------------------------

* Ember.js v3.16 or above
* Ember CLI v2.13 or above
* Node.js v10 or above


Installation
------------------------------------------------------------------------------

```
ember install mommy-look
```


Usage
------------------------------------------------------------------------------

```hbs
<MommyLook />
```
